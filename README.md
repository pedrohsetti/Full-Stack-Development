# Full-Stack Software Development

This repository contains the learning diary and crash course files for the Full-Stack Software Development module at Lappeenranta University of Technology. The diary documents the learning process and personal reflections on the skills acquired by following the crash courses on Next.js, MongoDB, Express, and React, as well as the development of a personal project using the MERN-stack.

You can find the project files in it's dedicated repository: [Reserva](https://github.com/pedrohsetti/Reserva)
